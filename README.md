# Proyecto 2025

Este es mi proyecto final desarrollado en Laravel para el programa ADSO del SENA (2025).

## Descripción
Este proyecto tiene como objetivo demostrar los conocimientos adquiridos durante la formación, aplicando buenas prácticas de desarrollo web con Laravel.

Incluye:
- Página de inicio
- Página Acerca del proyecto
- Página de contacto con formulario visual
- Layouts organizados y reutilizables

## Tecnologías utilizadas
- Laravel
- PHP
- Blade
- Tailwind CSS
- MySQL
- Git y GitHub

## Instalación
1. Clonar el repositorio
2. Copiar el archivo `.env.example` y renombrarlo a `.env`
3. Configurar la base de datos en el archivo `.env`
4. Ejecuta los comandos
